Projects

Aequatus

Aequatus is an open-source homology browser developed with novel rendering approaches to visualise homologous, orthologous and paralogous gene structures from Ensembl Compara and Ensembl Core databases.

Contact: For any query regarding Aequatus please mail at: tgac.browser@tgac.ac.uk

TGAC Browser

TGAC Browser is a new open-source Genomic Browser developed to visualise genome annotation from Ensembl Database Schema.

Contact: For any query regarding Aequatus please mail at: tgac.browser@tgac.ac.uk
